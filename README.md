# CatatUangku! - Modern Money Tracker

Aplikasi Flutter untuk mencatat pemasukan dan pengeluaran dengan tampilan UI yang elegan, animasi yang halus, dan grafik interaktif. Membantu Anda mengelola keuangan pribadi dengan cara yang simpel namun bertenaga.

## ✨ Fitur Utama

- 🔐 **Autentikasi Pengguna**: Login dan Register menggunakan Firebase Auth untuk keamanan data Anda.
- ☁️ **Cloud Sync**: Sinkronisasi data transaksi secara real-time dengan Firebase Firestore.
- 📊 **Grafik Interaktif**: Visualisasi pemasukan dan pengeluaran menggunakan bar chart dengan gradient warna yang menarik.
- 🔄 **Animasi Premium**: Transisi layar dan elemen UI yang smooth menggunakan kurva animasi modern.
- 🗂️ **Kategori & Filter**: Kelola transaksi berdasarkan kategori dan filter data berdasarkan bulan/tahun.
- 📝 **Manajemen Transaksi**: Tambah, edit, dan hapus transaksi dengan antarmuka yang intuitif.
- 📱 **Desain Liquid Glass**: UI modern yang bersih dengan fokus pada kejelasan visual dan pengalaman pengguna yang premium.

## 🛠️ Teknologi yang Digunakan

- **Framework**: [Flutter](https://flutter.dev)
- **Language**: [Dart](https://dart.dev)
- **Backend**: [Firebase](https://firebase.google.com) (Auth & Firestore)
- **Local Database**: SQLite (untuk caching dan offline support)
- **UI Components**: Curved Navigation Bar, Fl Chart, etc.

## 🚀 Memulai

### Prasyarat
- Flutter SDK (versi terbaru)
- Dart SDK
- Android Studio / VS Code
- Akun Firebase (untuk setup backend)

### Instalasi

1. **Clone repository:**
   ```bash
   git clone https://github.com/ilhamjagad/CatatUangku-MoneyTracker.git
   ```

2. **Masuk ke direktori proyek:**
   ```bash
   cd CatatUangku-MoneyTracker
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Konfigurasi Firebase:**
   Ikuti langkah-langkah di [PANDUAN_FIREBASE.md](PANDUAN_FIREBASE.md) untuk menghubungkan proyek ini dengan Firebase Console Anda.

5. **Jalankan aplikasi:**
   ```bash
   flutter run
   ```
