# CatatUangku! - Money Tracker

Aplikasi Flutter untuk mencatat pemasukan dan pengeluaran dengan tampilan UI elegan, animasi smooth, dan grafik interaktif.  
Tujuan utama aplikasi ini adalah membantu pengguna mengelola keuangan pribadi dengan cara yang simpel tapi powerfull.

## ✨ Features
- 📊 **Grafik Keuangan**: Bar chart dengan gradient warna untuk income & expense.
- 🔄 **Animasi Halus**: Transisi grafik dan tampilan data yang smooth.
- 🗂️ **Filter Data**: Bisa filter transaksi berdasarkan tanggal, bulan, atau kategori.
- 📝 **Manajemen Transaksi**: Tambah, edit, dan hapus transaksi dengan mudah.
- 📱 **UI Modern**: Desain clean dengan fokus pada visual clarity.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (versi terbaru)
- Dart
- Emulator atau device Android/iOS

### Installation
1. Clone repository:
   ```bash
   git clone https://github.com/ilhamjagad/CatatUangku-MoneyTracker.git

2. Masuk ke folder project:
   ```bash
   cd CatatUangku-MoneyTracker

3. Install dependencies:
   ```bash
   flutter pub get

4. Jalankan aplikasi:
   ```bash
   flutter run
